🧾 Step 1: Schema Design

Create or update the `subscriptions` table with the following fields:

```sql
ALTER TABLE subscriptions
ADD COLUMN name TEXT NOT NULL,
ADD COLUMN amount DECIMAL(12,2) NOT NULL,
ADD COLUMN billing_cycle TEXT CHECK (billing_cycle IN ('monthly', 'yearly', 'quarterly', 'bi-annual')),
ADD COLUMN next_due_date DATE,
ADD COLUMN description TEXT,
ADD COLUMN reconciled BOOLEAN DEFAULT FALSE,
ADD COLUMN subscription_type TEXT CHECK (subscription_type IN ('self', 'client')) DEFAULT 'self',
ADD COLUMN client_id UUID REFERENCES clients(id),
ADD COLUMN employee_id UUID REFERENCES employees(id),
ADD COLUMN category_id UUID REFERENCES categories(id),
ADD COLUMN payment_source_id UUID REFERENCES payment_sources(id),
ADD COLUMN organization_id UUID REFERENCES organizations(id) NOT NULL,
ADD COLUMN created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP;
