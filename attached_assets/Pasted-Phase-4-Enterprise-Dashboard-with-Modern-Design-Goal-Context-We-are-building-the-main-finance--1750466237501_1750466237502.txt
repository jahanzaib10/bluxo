Phase 4: Enterprise Dashboard with Modern Design
🎯 Goal & Context
We are building the main finance intelligence dashboard at /dashboard, used by founders and CFOs to track real-time performance across income, expenses, net profit, and subscriptions.

All financial data (income, expenses, recurring, categories, clients) is already linked and scoped to each organization.

We want this dashboard to:

Be highly visual and modern

Prioritize clarity, performance, and business utility

Look like a polished, investor-ready dashboard
